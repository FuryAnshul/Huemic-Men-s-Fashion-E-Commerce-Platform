# Huemic — Men's Fashion E-Commerce Website

A frontend e-commerce website for men's clothing, built with HTML, CSS, 
JavaScript and Bootstrap 5. Inspired by the design and feel of Snitch.so 
— one of India's top men's fashion brands.

---

## About The Project

Huemic is a men's fashion shopping website focused on clean UI and smooth 
user experience. The design takes inspiration from Snitch — modern layout, 
bold typography, category-based browsing, and a premium streetwear feel.
Built as a frontend project to practice real-world e-commerce UI development.

---

## Pages Built

- Home Page — Hero banner carousel, New Drops, Trending, Seasonal Collections
- Product Detail Page — Multi-image viewer, size selection (S to XXL)
- Search Page — Search bar with popular suggestions
- Cart Page — Cart layout with offcanvas support
- Login / Sign Up Page — OTP-based email login UI
- Admin Panel — Product upload forms with size and image options

---

## Frontend Features

- Inspired by Snitch.so design style
- Fully responsive on mobile, tablet and desktop
- Bootstrap 5.3 grid and components
- Sticky navigation bar with search, cart and profile icons
- Hamburger sidebar menu with category accordion
- Auto-rotating hero banner with offer messages
- New Drops vs Trending toggle using JavaScript
- Seasonal collection tiles with hover zoom effect
- Size selector buttons (S, M, L, XL, XXL)
- Floating rewards/loyalty button
- Newsletter signup in footer
- Social media links (Instagram, Facebook, YouTube, Twitter)
- Custom fonts — Futura and Twentieth Century for brand feel
- Bootstrap Icons throughout the UI

---

## Tech Stack

- HTML5 — Page structure
- CSS3 — Styling and animations
- JavaScript — Toggle effects, interactivity
- Bootstrap 5.3 — Responsive layout and UI components
- Bootstrap Icons — Icon library
- PHPMailer + PHP + MySQL — Basic backend for OTP login

---

## Inspiration

This project is inspired by Snitch.so — a popular Indian men's fashion brand 
known for its clean, modern, and mobile-friendly shopping experience. The goal 
was to replicate a similar look and feel while building original UI components.

---

## How to Run

1. Install XAMPP
2. Place project folder in: C:/xampp/htdocs/Huemic-Men-s-Fashion-E-Commerce-Platform/
3. Start Apache and MySQL
4. Visit: http://localhost/Huemic-Men-s-Fashion-E-Commerce-Platform/Huemic.php
